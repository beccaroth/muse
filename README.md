A personal creative projects dashboard for organizing ideas and tracking progress.

## Features

- **Kanban Board** — Drag-and-drop projects by Priority or Status
- **Table View** — Sort and browse all projects at a glance
- **Seeds Inbox** — Capture ideas and promote them to full projects
- **Progress Tracking** — Visual progress bars for each project

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Supabase (PostgreSQL)
- TanStack Query & Table
- dnd-kit
- Tailwind CSS + shadcn/ui
